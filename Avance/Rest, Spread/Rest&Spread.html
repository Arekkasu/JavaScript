Rest & Operador Spread:

Rest:

Tal como su nombre dice, se trata del Resto de datos de alguna variable, funciones, arrays, ETC. Para poder utilizar el resto es importante utilizar los tres puntos ...Variable, claramente despues del los tres puntos se le puede agregar cualquier nombre de variable al resto

Operador Spread:

Permite a un elmento iterable tal como un arreglo (array) o cadena ser expandido en ligares donde cero o mas argumentos (funciones) o elementos (arrays literales) son esperados, o aun objeto ser expandido en lugares donde cero o mas pares de valores clave (obhetos literales) son esperados.

IMPORTANTE:

El usar el rest con objetos o arrays anidados provoca el siguiente problema, el cual es que si se modifica la variable rest que tien el array u objeto que sea anidado este afectara la variable original y el rest
