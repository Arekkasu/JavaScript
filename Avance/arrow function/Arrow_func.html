Arrow Functions:

Una expresion de arrow function es una alternativa compacta a una expresion de la funcion tradicional, pero es limitada y no se puede utilizar en todas las situaciones.

Diferencias y limitaciones:

  • No tiene propios enlaces a this o super y no se debe usar como metodos
  • No tiene argumentos o palabras claves New.taget
  • No apa para los metodos Call, apply y bind, que generalmente se basan en estableces un ambito o alcance
  • No se puede utilizar como Constructor
  • No se puede utilizar Yield dentro de su cuerpo
