Datos Compuestos:

Los datos compuestos son los siguientes, Funciones, Objetos, Arrays y Clases

Funciones

Las funciones son un bloque de codigo propio, siendo diferente al entorno global, estos pueden recibir parametros o puede devolver valores.

Lo bueno de usar una funcion es que no se hace repeticion de codigo simplemente se realiza y se puede llamar poniendo el nombre de la funcion, en casa que tenga parametros se ponen en parentesis si no tiene solo se dejan vacios, seria asi Sumar(50, 40) o Saludad(), tambien una variable puede tener una funcion como valor; No toda funcion requiere un return y las funciones solo ejecutaran el primer return que aparezca en el codigo

Funcion Expresada VS Declarada:


Declarada

Expresada o Anonima

Funciones declaradas pueden ser activadas hasta antes del bloque que se creo la funcion, pero, Funcion anonima/expresada sucede que deber ser llamada si o si despues de ser nombrada

Arrays (Arreglos)

los arrays es un cojunto de datos ordenados por posiciones, asociados en una sola variable, los arrays es como una Lista

los arrays se leen desde 0 al iniciar a contar los elementos, es decir si un array tiene 5 elementos al contarlos seria del 0 al 4, siguen siendo los 5 elementos pero no se cuenta desde el 1.
Un array tambien puede tener otro array adentro, otra cosa que se puede hacer con un array es poder establecerlo sin que sea en una sola linea sino que sea multiples lineas Siempre y cuando esten separadas por comas.

Para hacer llamado de un array simplemente se escribe el nombre de la variable que le asignaste al Array

Si no quieres ver toda la lista de un array sino que quieres solo ver un elemento de este, se hace poniendo el nombre del array y entre corchetes el ID del elemento.

En la imagen se mosotro como usarlo en un array dentro de un array, pero, para un elemento simplementeseria array[id], pero, hay dos cosas que se tienen que saber para que no haya error en el codigo, para saber si un elemento pertenece se usa array.includes() sabiendo esto, para saber la ID del elemento se usa arry.indexOf()

Methods Arrays:

Array.length:

Al usar este metodo devolvera la longitud del array

Array.push(elemento):

Este metodo sirve para agregar algun elemento al array

Array.pop():

Este metodo sirve para porder elminar el ultimo elemento de un array

Array.splice( id, elementos a borrar):

Sirve para eliminar un elemento determinado de un array, siendo primero poner el ID del elemento y despues de la coma poner cuantos elementos borrara hacia la derecha, siempre y cuando se ponga 2 o mas.

Array.forEach(function (elemento, index){}):

Es una funcion que recorrera todos los items de un array, agregando 2 parametros que cumpliran como la id y el elemento del array

Objetos (Objects)

Los obejetos son una coleccion de propiedad y cada propiedad esta asociada entre una clave y un valor. El caso en que una propiedad tenga una funcion como valor sera conocida como un metodo

Para poder Llamar un objeto simplemente se pone el nombre y mostrara todo lo que contiene, pero si necesita saber el valor de una clave se pone objeto.clave; en caso que exista una clave con valor array se haria de esta forma objeto.clave[], en el objeto de imagen, la clave pasatiempos tiene como valor un array por lo cual si quieremos ver algun elemento del array se usaria de esta manera, user.pasatiempos[2], siendo videojuegos el valor lanzado

Para agregar una propiedad al objeto se pone objeto[clave] = valor

Metodos:

Object.keys:

Con este metodo podemos obtener todos las claves pero sin su valor de un objeto

Object.values:

Muestra solo los valores de los claves

Objeto.hasOwnProperty(clave):

Con este metodo sirve para conocer si la propiedad esta en el objeto

Operadores Comparacion:

Operador Descripcion Ejemplo
== Es una igualdad de valor sin importar su tipo de dato. 10 == '10' (True)
=== Estrictamente igual evalua el valor y que sean tambien su mismo tipo de dato. 10 === '10' (False)
!= Es una desigualdad de valor sin importar su tipo de dato. 2 != '2' (false)
!== Estrictamente desigual evalua el valor y que sean tambien desigual su tipo de dato. 2 !== '2' (True)
- Resta 5-6 = 1
