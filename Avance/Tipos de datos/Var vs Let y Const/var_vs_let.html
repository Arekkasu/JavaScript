Var, Let y Const

Var, se usa para declarar las variables en JS, donde Var es en ambito global del codigo, mientras que Let es mas adecuado para determinado bloques y const se usa como una constante, osea, su valor no se puede redeclarar, PERO, con los arrays y objetos si se puede ir 'Modificando'.
