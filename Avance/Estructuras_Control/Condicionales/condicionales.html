Condicionales:

Es un conjunto de comandos que se ejecutan si una condicion es verdadera. en JS estan presentes tres tipos de condicionales:

If... Else

Se inicia con un IF para saber que la condicion es verdadera, en que caso que no sea verdadera, seguira con else en caso que sea falso.
Si la condicion evalua mas de dos eventos, se usa else IF.

Operador Ternario

Es como el if.. else pero Simplificad, es decir, se una en usa sola linea, su estructura es la siguiente.
condicion ? valor 1 : valor 2

Operador Switch

Es un condicional que permite hacer multiples operaciones, y tomar decisiones en funcion de distintos estados de las variables.
Evalua una expresion, comparando el valor de esa expresion con una instancia llamada Case, y se ejecuta las declaraciones asosciadas a ese Case.

Ciclos

Sirve para ejecutar un cico que se ejecuta un codigo indefinidamente.

While:

Ejecuta un ciclo de sentencias mientras se cumpla una condicion

Do while:

Similar al while, solo que evalua la condicion al final.
